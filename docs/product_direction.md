# Supertime

Supertime is a calm, human-first platform for intentional conversations.

It is not a productivity SaaS tool.
It is not a freelancer marketplace.
It is not a social media app.

Supertime exists to help people spend meaningful time with creators, artists, thinkers, experts, and emotionally expressive humans.

The platform should feel:
- calm
- open
- transparent
- human
- breathable
- emotionally safe
- internet-native
- culturally alive

The product should avoid:
- corporate SaaS aesthetics
- dopamine-driven UX
- clutter
- excessive gamification
- overly dense dashboards
- growth-hack design patterns

---

# Core Philosophy

Technology should disappear into the background.

The focus should always remain on:
- people
- conversations
- presence
- intention
- emotional connection

The product should feel more like a public creative space than a software dashboard.

Think:
- Are.na
- early internet forums
- quiet creative communities

Not:
- LinkedIn
- Fiverr
- enterprise software

---

# MVP Direction

We are simplifying the experience, not removing capabilities.

Current focus:
- improve clarity
- improve emotional experience
- reduce cognitive load
- improve mobile-first UX
- simplify interactions
- improve visual rhythm and spacing

---

# Core MVP Features

The platform currently supports:
- profiles
- booking
- payments
- audio/video calls
- scheduling
- creator sessions

We are NOT removing these features.

We are simplifying how users experience them.

---

# Mobile First

All new design decisions should prioritize mobile-first layouts and interactions.

The interface should:
- feel lightweight
- load quickly
- avoid visual heaviness
- minimize friction
- work comfortably on older devices

---

# Design Principles

Use:
- generous spacing
- restrained typography
- minimal color usage
- low visual noise
- soft hierarchy
- calm interaction flows

Avoid:
- heavy gradients
- flashy animations
- excessive shadows
- startup cliché aesthetics
- unnecessary UI chrome

---

# Typography Direction

Typography should feel:
- neutral
- warm
- readable
- understated
- emotionally calm

Avoid overly decorative or luxury editorial typography.

Current exploration:
- Inter
- Instrument Sans
- Satoshi
- modern humanist sans-serif systems

Typography should support humans, not dominate attention.

---

# Product Direction

Supertime is evolving toward:
- intentional human connection
- creator interactions
- emotionally meaningful conversations
- lightweight communication infrastructure

Long term:
- native realtime communication
- instant creator interaction
- open communication infrastructure

But current focus is:
- simplicity
- usability
- emotional clarity
- meaningful interactions

---

# Important

Do not overengineer.

Do not redesign for aesthetics alone.

Do not add features without strong product reasoning.

Prioritize:
- clarity
- emotional tone
- simplicity
- coherence
- human experience
